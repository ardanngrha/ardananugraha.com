## Overview

FlowState is a native macOS application designed to help users achieve deep focus and maintain productivity. It combines time-blocking techniques with distraction management tools to create an optimal work environment.

## Key Features

- **Focus Sessions**: Customizable Pomodoro-style work intervals
- **Distraction Blocking**: Temporarily block distracting websites and applications
- **Progress Tracking**: Detailed analytics on focus sessions and productivity trends
- **Native Integration**: Seamless macOS integration with notifications and menu bar

## Technologies Used

- **Language**: Swift 5.0+
- **Framework**: SwiftUI, AppKit
- **Data**: Core Data for local storage
- **Architecture**: MVVM pattern
- **Platform**: macOS 12.0+

## Development Highlights

- Implemented complex state management for focus sessions
- Created custom UI components following Apple's design guidelines
- Integrated with macOS system APIs for app/website blocking
- Built comprehensive analytics dashboard
