# Forum API

## Overview

A comprehensive backend API for a forum application featuring user management, post creation, commenting system, and real-time notifications. Built with modern Node.js practices and scalable architecture.

## Key Features

- **User Management**: Registration, login, profile management
- **Post System**: Create, read, update, delete posts with rich content
- **Comment System**: Nested comments with threading support
- **Authentication**: JWT-based secure authentication
- **Real-time Updates**: WebSocket integration for live updates

## Technologies Used

- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
- **Authentication**: JWT, bcrypt
- **Testing**: Jest, Supertest
- **Documentation**: Swagger/OpenAPI

## API Endpoints

- User authentication and profile management
- Post CRUD operations with pagination
- Comment system with nested replies
- Vote/rating system for posts and comments
