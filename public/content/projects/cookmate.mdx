# CookMate

## Overview

CookMate is an intelligent cooking companion that leverages machine learning to provide personalized recipe recommendations, meal planning, and cooking guidance. The app learns from user preferences and dietary restrictions to suggest optimal meals.

## Key Features

- **Smart Recommendations**: ML-powered recipe suggestions based on preferences
- **Meal Planning**: Intelligent weekly meal planning with nutritional balance
- **Ingredient Recognition**: Camera-based ingredient identification
- **Shopping Lists**: Automated grocery list generation
- **Cooking Timer**: Multi-timer support with voice notifications

## Technologies Used

- **Mobile**: Android (Kotlin), Jetpack Compose, Room Database
- **Backend**: FastAPI, Python, TensorFlow
- **Cloud**: Google Cloud Platform, Cloud Storage, Cloud Functions
- **Machine Learning**: Recommendation algorithms, Computer Vision
- **Database**: PostgreSQL, Cloud Firestore

## Technical Implementation

- Implemented MVVM architecture with Repository pattern
- Created efficient offline-first data synchronization
- Built scalable microservices backend architecture
- Integrated real-time ML inference for ingredient recognition
