# Second Project

This is a full-stack web application that demonstrates modern development practices and technologies.

## Overview

The Second Project is a task management application that helps users organize their daily activities and collaborate with team members. Built with a focus on performance and user experience.

## Features

- User authentication and authorization
- Real-time task updates
- Team collaboration tools
- Responsive design
- Dark/light theme support
- File attachment handling

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT, bcrypt
- **Real-time**: Socket.io
- **Deployment**: Vercel, MongoDB Atlas
